How It Evolves

Week 1 — Learning (C)

Companion panel is front-and-center with guided walkthroughs. Sidebar shows Quick Start progress. Panel opens automatically on launch.

Month 1 — Command Center (B)

Quick Start is done. Panel becomes a contextual dashboard — connector health, recently used skills, suggested guides. Opens on click, not automatically.

Month 3+ — Invisible (A)

Panel stays closed. Sidebar is collapsed. PM Code surfaces only via notifications ("Jira token expiring") or when the user explicitly opens it. The agent just works.
