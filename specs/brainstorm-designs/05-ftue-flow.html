Key Design Decisions

Every step teaches VS Code

Step 1 teaches sidebars. Step 3 teaches the chat. Implicit learning, not a VS Code tutorial.

Every step is skippable

Non-blocking. Skipped steps appear in the sidebar's Quick Start section for later.

Auto-detect, don't ask

Roo Code is detected automatically. No "which AI provider?" question.

Connector setup is form-based

Clicking "Jira" opens a guided form: paste your URL, paste your token, test, done.
