## Try Talking to Your AI

Send your first real prompt to see PM Code in action. If you connected a tool in the previous step, Roo will use it to pull real data.

### Try one of these

**If you connected Jira:**
> "What are the open issues assigned to me this sprint?"

**If you connected GitHub:**
> "Summarize the most recent PRs in our repo"

**If you connected any tool:**
> "Give me a standup summary of recent activity"

**Without any connections:**
> "Help me write a PRD for a user onboarding flow"

### Tips for effective prompts

- **Be specific** — "Show me P1 bugs from the last week" works better than "Show me bugs"
- **Name your tools** — "Check Jira for..." tells Roo which connector to use
- **Ask for structure** — "Organize by priority" or "Format as a table" helps get useful output
- **Iterate** — Follow up with "Make it shorter" or "Add acceptance criteria" to refine

### Use Skills for common workflows

Instead of writing prompts from scratch, try **Skills** — pre-built workflows for common PM tasks. Open the PM Code sidebar and click **Skills** to browse them. Each skill comes with ready-to-use prompts.
