How It Works

Home → List → Detail

Three states: home (buttons), list (skills/connectors/guides), detail (opens center panel). Back button returns to home.

Search takes over

Typing in search replaces everything below with grouped results. Clear search (✕ or Escape) returns to previous state.

All WebviewView

Single WebviewViewProvider manages all sidebar states. Full layout control, custom styling, richer content than TreeView allows.
