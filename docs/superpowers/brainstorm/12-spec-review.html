Key Features

C → B → A Progression
Learning → Command Center → Invisible. Auto-transitions based on usage signals.
Zero JSON / Zero Terminal
Form-based config for everything. Advanced escape hatch for power users.
Background System Check
Installs Xcode CLT, Node, Python, gh CLI during FTUE. Human-friendly progress.
Interactive Prompt Injection
"Send to Roo →" buttons on skills and guides inject prompts into Roo Code.
Cross-Linked Content
Every panel shows related Commands, Connectors, Skills, and Guides.
Rollback / Undo
Config snapshots on every change. "Reset to working state" for when things break.
VS Code Walkthroughs
Native onboarding API for FTUE + Projects/Context guide. Rich panels for the rest.
Auto-Detection
Detects Roo Code, existing MCP config, workspace type. No "which provider?" questions.
